# Full Stack Web Application

A full-stack web application built using **React.js** for the frontend and **Node.js with Express.js** for the backend.  
This project demonstrates clean UI, REST API integration, and basic full-stack architecture.

---

##  Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript (ES6)

### Backend
- Node.js
- Express.js

- ## Database
- PostgreSQL

---

##  Features

- Responsive user interface
- Clean and simple design
